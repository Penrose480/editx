# EDITX

A simple, minimal, efficient text editor. 

## INSTALLATION

Clone or copy symlinks invidually. After do the following:

```bash
# Go into cloned target
cd [INSERT CLONED DIRECTORY HERE]
# make
make
```


## USAGE

### Run the editor 
```bash
./editx
```
### Edit a file:
```bash
./editx [INSERT FILENAME HERE]
```

### Keybinds

CTRL + q: quit
CTRL + s: save 

## PHILOSOPHY

Bloated software lurks around every corner. I combat these issues with simple, clear, minimal code.
